# app-questionaire-angular

### Project URL

https://app-questionaire-angular.vercel.app/

### Set up

Open terminal

> git clone git@github.com:abdullah-sust/app-questionaire-angular.git

> cd app-questionaire-angular

> npm install

> ng serve
